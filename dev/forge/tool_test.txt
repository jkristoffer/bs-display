## Summary of forge.py

**Forge MVP** is an AI-First Orchestrator - a Python CLI tool that interfaces with Claude AI for automated file generation and codebase analysis.

### Core Functionality
- **Single file creation**: Generate individual files based on text prompts
- **Multi-file generation**: Create multiple related files using a structured format
- **Codebase analysis**: Analyze existing code with context awareness tools

### Key Components

**Data Models** (using Pydantic):
- `PromptRequest`: User input validation
- `FileSpec`: File path and content specification  
- `ClaudeResponse`: Claude's response wrapper
- `MultiFileResponse`: Multiple file container

**Core Functions**:
- `call_claude()`: Interface with Claude CLI via subprocess
- `parse_multi_file_response()`: Parse Claude's multi-file format responses
- `strip_markdown_formatting()`: Clean Claude responses of markdown
- `detect_tool_calls()`: Identify tool requests in Claude responses
- `read_file_content()` / `list_directory_contents()`: File system operations

**CLI Commands**:
1. `create-file`: Generate single file from prompt
2. `create-files`: Generate multiple related files  
3. `analyze`: Perform codebase analysis with context awareness

### Technical Features
- Comprehensive error handling and logging
- Pydantic validation for all inputs/outputs
- Subprocess integration with Claude CLI
- Tool call detection (for future automation)
- Markdown cleaning for clean file output
- Directory creation with proper permissions

The tool serves as a bridge between human prompts and Claude AI, enabling automated code generation and analysis workflows.
