## Forge Project Overview

**Forge** is an **AI-First Orchestrator** - a Python CLI tool that uses Claude AI to generate files and analyze codebases. The main script `forge.py` is a 463-line application built with Typer and Pydantic.

### Core Files:
- **`forge.py`** - Main orchestrator CLI with 3 commands (create_file, create_files, analyze)
- **`requirements.txt`** - Python dependencies 
- **`forge.log`** - Application logs
- **Test files** - Various Python scripts and JSON configs for testing

### Key Features:
1. **Single file generation** - `create_file` command generates one file from a prompt
2. **Multi-file generation** - `create_files` command creates multiple related files using `FILE:` markers
3. **Codebase analysis** - `analyze` command with context-aware tools that can detect tool calls

### Test Output Structure:
The `test-output/` directory contains generated React components with proper structure:
- TypeScript React components (`.tsx`)
- SCSS modules (`.module.scss`) 
- Test files (`.test.tsx`)
- Index exports (`index.ts`)

This appears to be an MVP for automating code generation workflows using Claude CLI, focusing on creating properly structured frontend components and performing codebase analysis.
